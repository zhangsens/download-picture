#pixiv download：

>从pixiv上获取图片并下载

食用方法：

```
node index.js
```

或者

```
npm start
```

目前get每日插画rank

图片直接download到picture文件夹内

单图文件名为id_0.png,

多图文件名为id_0.png,id_1.png,id_2.png,id_3.png......

(现在保存为.png，不影响查看)

基本功能已经完善
